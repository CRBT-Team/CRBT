Clembs API

This API is for now in beta, and will remain as such until the full release, planned alongside the release of CRBT v11.

GitHub repository (soon) - Discord server - Website

Endpoints:

CRBT:

GET /crbt/meaning: Acronyms for CRBT
GET /crbt/command?name=<command name (e.g. help, crbt, pfp)>: Gives information for a command.
Coming in v11: a real CRBT API.

Other:

GET /other/anime?name=<anime name>: Gives the first result for your query on Kitsu.io.
GET /other/color/<HEX color>: Returns an image of the specified HEX color code.
GET /translate?text=<text to translate>&to=<target language (optional)>&from=<Source language (optional)>: Translates text to any language.
