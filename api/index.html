Discord server - Website

Endpoints:

CRBT:

GET /crbt: Retrieves info on CRBT & its latest version.
GET /crbt/meaning: wHaT dOeS cRbT mEan?????????2?
GET /crbt/command/<command name (e.g. help, crbt, pfp)>: Gives all of a specified command's property.

Other:

GET /other/anime/<anime name>: Searches a specified anime on kitsu.io.
GET /color/<HEX color>: Gives you an image of the specified HEX color code.
